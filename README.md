## Project Overview

This project focuses on predicting customer churn for a telecom company using Machine Learning techniques.

The objective is to identify customers who are likely to discontinue services and provide business recommendations for customer retention.

---

## Dataset

Dataset Used:

WA_Fn-UseC_-Telco-Customer-Churn.csv

Source:
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

Records: 7043

Features: Customer demographics, services subscribed, contract information, billing information, and churn status.
